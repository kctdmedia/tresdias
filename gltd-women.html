    Setup Guide for Leadership

    1

    Make a copy of the template sheet: Click here to copy the template ↗ — it has all 5 tabs with correct headers pre-filled.

    2

    Publish each tab as CSV: File → Share → Publish to web → select the tab → choose Comma-separated values (.csv) → Publish → copy the URL → paste it above. Repeat for all 5 tabs.

    3

    Edit the Settings tab to set your weekend title, verse, theme colors, venue info, fee amounts, payment links, and doc links. Every row is a Key + Value pair — just update the Value column.

    4

    For photos: Upload to Google Drive → Share → “Anyone with the link” → copy the link → paste in the Photo column of the Roster tab. The app converts it automatically.

    5

    Share the Sheet with your leadership team as Editor. They edit rows directly — no app login required. Hit Refresh in the app to pull the latest changes immediately.

    Settings tab — Key names (edit the Value column):

    Weekend: weekend_title · weekend_verse · weekend_tagline · weekend_colors · weekend_symbols · weekend_songs · group_name · group_number

    Venue: venue_name · venue_address · venue_map_url · airport_1 · airport_2 · airport_3

    Fees: fee1_label · fee1_amount · fee1_note · fee1_apply_url · fee1_venmo_url · fee1_due · fee1_mail
    (repeat for fee2_, fee3_, fee4_)

    Docs: doc1_label · doc1_url · doc2_label · doc2_url … up to doc6_

    Meetings tab columns: Title · Date · Time · Location · Type · AgendaURL
    Roster tab columns: Name · Role · Day · Status · Photo · Notes
    Prayer tab columns: Name · Request · Urgency · Date · Anonymous
    Palanca tab columns: From · For · Message · Delivery · Date
